Wanneer snappen is ingeschakeld, dan “plakt” een punt dat je tekent of bewerkt automatisch vast aan een punt of lijn van een ander object dat al op de kaart staat. Hierdoor sluiten objecten exact op elkaar aan. Snappen kan gebruikt worden bij het tekenen van nieuwe objecten en bij het aanpassen van bestaande objecten.

Instelmogelijkheden voor snappen:

  • Snapafstand: de afstand (in pixels) waarbinnen een lijn vastklikt op een object
  • Snappen op meerdere lagen
  • Snappen op randen (edges), hoekpunten (vertices), snijpunten (intersections) of segmenten
