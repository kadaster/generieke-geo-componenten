Deze functie wordt vaak gebruikt om van achtergrondkaart te wisselen (bijvoorbeeld met een luchtfoto), maar kan ook gebruikt worden om een complete verzameling van kaartlagen te wisselen.

Om de knoppen van de buttons te koppelen aan de kaartlagen, is het belangrijk om de DatasetSwitcherButton dezelfde naam te geven als de themeName uit de datasetSwitcherConfig. Deze naam wordt ook gebruikt bij het tonen van de knop.

Het stylen van de buttons is mogelijk door het overschrijven van de css classes .ggc-ds-switcher-label, .ggc-ds-switcher-label-img, .ggc-ds-switcher-label-txt en .ggc-ds-switcher-radio-buttons. Bekijk de SCSS voor een voorbeeld hiervan.
